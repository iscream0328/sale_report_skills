# 유사/반대 매칭 및 영업 산출물 생성 기준

## 1. 매칭의 기본 관점

이 워크플로우의 추천은 "예쁜 이미지 검색"이 아니라 영업 제안의 논리를 만드는 일이다.

- 유사 추천: 브랜드가 이미 잘하고 있는 방향과 자연스럽게 연결되는 포트폴리오
- 반대/보완 추천: 브랜드에 부족한 컷 역할, 톤, 구도, 커머스 기능을 제안할 수 있는 포트폴리오

"반대"는 브랜드와 맞지 않는 이미지를 뜻하지 않는다. 브랜드가 아직 보여주지 않은 가능성을 설득할 수 있는 대비 방향이다.

## 2. 브랜드 이미지 요약 벡터

브랜드 URL에서 수집한 상품 10장과 캠페인 10장을 각각 요약한다.

상품 이미지 요약:

- 주요 상품군
- 상품 전달력
- 배경/조명 일관성
- 착용/상세/정물 비율
- 구매 판단에 도움 되는 정보가 충분한지

캠페인 이미지 요약:

- 브랜드 무드
- 모델/장소/세트 사용 방식
- 시즌성
- 그래픽/타이포 활용
- SNS 후킹 가능성

통합 브랜드 요약:

- `brand_tone`
- `strong_visual_assets`
- `missing_cut_types`
- `commerce_gaps`
- `proposal_angles`

## 3. 유사 추천 점수

유사 추천은 브랜드가 이미 가진 이미지와 이어 붙였을 때 자연스럽게 설득되는 포트폴리오를 찾는다.

권장 가중치:

```text
similar_score =
  0.35 * image_embedding_similarity
+ 0.20 * visual_tag_overlap
+ 0.15 * cut_type_match
+ 0.15 * commerce_role_match
+ 0.10 * brand_tone_match
+ 0.05 * project_group_diversity_bonus
```

설명 문장에는 아래 근거를 반드시 포함한다.

- 어떤 브랜드 이미지와 비슷한지
- 어떤 태그나 컷 역할이 겹치는지
- 그래서 어떤 제안 문장으로 이어질 수 있는지

## 4. 반대/보완 추천 점수

반대/보완 추천은 브랜드의 부족한 지점을 채울 수 있는 포트폴리오를 찾는다.

권장 가중치:

```text
contrast_score =
  0.30 * missing_cut_type_fit
+ 0.20 * visual_contrast_strength
+ 0.20 * commerce_gap_fit
+ 0.15 * portfolio_strength_for_angle
+ 0.10 * brand_fit_guardrail
+ 0.05 * project_group_diversity_bonus
```

`brand_fit_guardrail`이 낮은 이미지는 아무리 대비가 강해도 제외한다. 예를 들어 프리미엄 미니멀 브랜드에 너무 유치한 팝 컬러 컷을 추천하는 식의 어긋남을 막기 위함이다.

## 5. 다양성 제한

상위 5장이 같은 프로젝트나 같은 컷 유형으로 몰리면 제안이 약해진다.

추천 제한:

- 같은 `project_group_id`는 최대 2장
- 같은 `cut_type`은 최대 3장
- 유사 추천과 보완 추천이 같은 이미지를 공유하지 않음
- 보완 추천은 최소 2개 이상의 다른 제안 각도를 포함

## 6. 추천 카드 구성

각 추천 카드는 아래 정보를 가진다.

- 포트폴리오 이미지
- 추천 유형: 유사 또는 보완
- 매칭 점수
- 매칭 근거 2~3개
- 브랜드 이미지 참조 ID
- 제안 문장
- 사용 여부
- 담당자 메모

## 7. 담당자 선택 플로우

담당자는 추천 결과에서 실제 발송할 이미지를 고른다.

상태:

- `recommended`: 시스템 추천
- `selected`: 제안서에 사용
- `rejected`: 제외
- `needs_review`: 권리/브랜드명/문구 검수 필요

선택 화면에서 필요한 조작:

- 유사/보완 필터
- 컷 유형 필터
- 프로젝트 그룹 필터
- 선택 트레이
- 이미지별 메모
- 제안서 포함 순서 변경

## 8. 영업 메일 초안 구조

메일은 짧고 구체적이어야 한다.

권장 구조:

1. 브랜드 이미지를 실제로 봤다는 문장
2. 현재 이미지의 강점 1개
3. 우리가 보완할 수 있는 지점 1개
4. 선택한 포트폴리오와 연결되는 제안 1~2개
5. 미팅 또는 레퍼런스 공유 제안

금지:

- 브랜드 매출, 성과, 내부 전략을 단정
- 선택되지 않은 포트폴리오를 근거로 언급
- 이미지로 확인할 수 없는 내용을 과장

## 9. 제안서 초안 구조

제안서는 이미지 중심으로 간결하게 만든다.

### 1페이지: 브랜드 현재 이미지 요약

- 수집한 상품/캠페인 이미지 썸네일
- 현재 강점
- 보완 가능 지점

### 2페이지: 유사 포트폴리오

- 선택한 유사 포트폴리오 이미지
- 브랜드 이미지와 닮은 점
- 이 방향을 확장했을 때의 촬영/디렉팅 제안

### 3페이지: 보완 포트폴리오

- 선택한 보완 포트폴리오 이미지
- 현재 브랜드 이미지에서 부족한 컷 역할
- 새롭게 제안할 수 있는 컨셉

### 4페이지: 실행 제안

- 컨셉 2~3개
- 필요한 작업 범위: Visual Directing, Styling, Set Design, Short-form Guide 등
- 예상 산출물: 상세페이지 상단, SNS 키비주얼, 릴스 커버, 제품 디테일 컷 등
- 다음 단계: 20분 미팅 또는 레퍼런스 보드 공유

## 10. 품질 검수 체크리스트

- 상품 이미지 10장과 캠페인 이미지 10장의 분류가 납득 가능한가
- 유사 추천 5장이 브랜드 이미지와 실제로 연결되는가
- 보완 추천 5장이 브랜드에 무리 없이 제안 가능한가
- 같은 프로젝트 이미지가 과도하게 반복되지 않았는가
- 메일 문구가 선택 이미지와 연결되는가
- 제안서에 출처가 불명확한 이미지가 들어가지 않았는가
- 최종 발송 전 사람이 권리/표기/톤을 검수했는가
